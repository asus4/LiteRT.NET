# LiteRT Unity Utilities

Optional Unity helpers for [LiteRT](https://github.com/google-ai-edge/LiteRT): texture-to-tensor conversion, vision task base classes, and model loading utilities. Requires the core bindings package `com.github.asus4.litert`.

Part of [LiteRT.NET](https://github.com/asus4/LiteRT.NET). See the repository for documentation and examples.
