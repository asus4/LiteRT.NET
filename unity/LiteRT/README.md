# LiteRT

C# bindings for [LiteRT](https://github.com/google-ai-edge/LiteRT) with the prebuilt native runtime for Unity (macOS, Windows, Linux, iOS, Android).

Part of [LiteRT.NET](https://github.com/asus4/LiteRT.NET). See the repository for documentation and examples. Optional Unity helpers (texture-to-tensor conversion, vision task base classes) live in the companion package `com.github.asus4.litert.unity`.
